Piano patate série + calibration

Cette page HTML tourne sur le PC. Elle fait trois choses principales : 1) lire le port série de l'ESP32, 2) envoyer des commandes à l'ESP32, 3) générer les notes audio dans le navigateur.

Série déconnectée Audio inactif Calibration inconnue
Commande envoyée vers l'ESP32 :
- CALIBRATE

Messages série compris :
- TOUCHE 0 / RELACHE 0
- CALIBRATION_START / CALIBRATION_DONE
- CAL n GPIO x BASE y LOW z HIGH h
